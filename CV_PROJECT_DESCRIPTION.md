Fanorona Telo - Application de Jeu Stratégique avec Intelligence Artificielle

Janvier 2026 - En cours | Projet académique - Développeur solo

Gestion complète du cycle de jeu : Développement d'une application pour le jeu traditionnel malgache Fanorona Telo avec système de joueurs, mouvements validés et calculs de scores en temps réel.

Intelligence Artificielle & Minimax : Implémentation d'un système d'IA utilisant l'algorithme Minimax avec exploration configurable d'arbre de recherche et support du mode IA vs IA.

Interface & Géométrie Computationnelle : Développement d'une interface graphique interactive en Tkinter avec gestion géométrique des coordonnées et calculs de distances euclidiennes.

Stack : Python 3.8+ | Tkinter | NumPy | Shapely | GitHub

Outils & Méthodologie : GitHub | Docker | Agile/Scrum

Compétences : Algorithmique avancée (Minimax), programmation orientée objet, géométrie computationnelle, interfaces graphiques.
